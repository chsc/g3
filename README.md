g³
==

g³ is a rudimentary real time rendering engine written in Go.
Needs Go-SDL and Go-OpenGL to run.

### Features ###
 * Geomipmapping ([Article by Willem H. de Boer](http://www.flipcode.com/archives/article_geomipmaps.pdf))
 * to be continued ...
